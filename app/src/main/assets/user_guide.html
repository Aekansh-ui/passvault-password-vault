First Launch & Authentication

When you open Pass Vault you will see the lock screen with the app logo in the centre. The app cannot be entered without verifying your identity.

How to unlock

  1. Tap the logo card in the middle of the screen.
  2. Your device's biometric prompt will appear — use your fingerprint, face, or device PIN.
  3. Once verified, you are taken to the Home Screen.
Note: If you cancel or fail authentication, the app closes completely. This is intentional — Pass Vault cannot be accessed without a successful verification.
2

Home Screen

After authentication you land on the Home Screen. This is your main vault view showing all saved services.

What you see

  • Groups — Every website or service you save becomes a group (e.g. "Google", "Netflix"). Groups are listed alphabetically.
  • Red border on a group card — One or more passwords in that group are due for a change based on your reminder settings.
  • Search bar — Filters groups by name as you type.

Bottom navigation bar

3

Adding a Password

Tap the + button at the bottom of any screen to open the Add form.

Fields

FieldWhat to enter
URLThe website address (e.g. google.com). Pass Vault extracts the site name automatically to create or find the group.
IDYour email address or username for that account.
PasswordYour password. Tap the eye icon to show or hide it.

Generate a password

Tap GENERATE NEW to instantly create a strong 16-character random password mixing uppercase, lowercase, numbers, and symbols. Tap it again to get a different one.

Periodic Reminder (optional)

  1. Select YES under Periodic Reminder.
  2. Choose the unit: DAYS, WEEKS, or MONTHS.
  3. Choose the number (1 – 180) from the dropdown.

Saving

Tap ADD PASSWORD. Pass Vault creates a new group if the site doesn't exist yet, or adds the account to an existing group.

Duplicate detected? If you add an ID that already exists in the same group, Pass Vault takes you to the Update screen automatically instead of creating a duplicate.
4

Viewing a Saved Password

From the Home Screen, tap a group then tap an account row to open the detail view.

What you see

  • Date — When this password version was created.
  • Website — The URL you saved.
  • Email / Username — Your login ID.
  • Password — Hidden by default (shown as dots).

Revealing the password

  1. Tap the eye icon next to the password.
  2. Authenticate with biometric or PIN when prompted.
  3. The password is shown in plain text.
  4. Tap the eye icon again to hide it immediately.

Copying the password

Tap the copy icon next to the password. The password is copied to your clipboard and will automatically clear after 30 seconds. A message at the bottom of the screen confirms the copy.

Tip: You don't need to reveal the password to copy it — copying works even when the password is hidden.
5

Updating a Password

  1. Open the account detail screen.
  2. Tap UPDATE.
  3. Edit the URL, ID, password, or reminder settings as needed.
  4. Tap SAVE CHANGES.
Note: Each update creates a new password version. Your previous password is automatically saved in history — nothing is lost.
6

Password Version History

Pass Vault keeps up to 10 previous versions of each password. This lets you go back if you made a mistake or need to reference an old credential.

Switching versions

  1. Open the account detail screen.
  2. Tap the version chip in the top-right corner (e.g. v3 ▼).
  3. A dropdown lists all saved versions — tap any to view it.
Note: The Update and Delete buttons are only shown when you are viewing the latest (current) version.
7

Deleting a Password

  1. Open the account detail screen.
  2. Tap DELETE.
  3. Read the confirmation dialog — it explains exactly what will happen.
  4. Tap Confirm to proceed, or Cancel to go back.

What happens on delete

SituationResult
Older versions existCurrent version is removed; the previous version becomes active.
Only one version existsThe entire account is permanently deleted.
Last account in a groupThe account and the group are both deleted.
8

Groups

Groups are created and managed automatically — you do not create them manually.

  • When you add a password, Pass Vault uses the site name from the URL as the group name.
  • When you delete all accounts in a group, the group disappears too.
  • Tapping a group on the Home Screen shows all accounts inside it.
  • Each group screen has its own search bar to filter accounts by username.
Tip: To add a second account to an existing group (e.g. a second Google account), use the same URL. Pass Vault will place it in the same group automatically.
9

Periodic Reminders

If you set a reminder on a password, Pass Vault will alert you when it is time to change it.

How it works

  • The countdown starts from the last time the password was changed.
  • When the deadline is within 5 days, group and account cards show a red border.
  • A background check also sends a push notification to remind you.

Changing or disabling a reminder

  1. Open the account → tap UPDATE.
  2. Scroll to Periodic Reminder.
  3. Select NO to disable, or change the unit and value.
  4. Tap SAVE CHANGES.

Profile

Tap the person icon at the bottom right to open your profile. It displays your name, username, and photo.

Updating your profile

  1. Tap Update Profile.
  2. Edit your Display Name, Username, or tap the photo area to pick an image from your gallery.
  3. Tap SAVE.

Backup & Restore

Creating a backup

  1. Go to Settings → tap EXPORT BACKUP.
  2. Authenticate with biometric or PIN when prompted.
  3. Choose where to save the file on your device (any folder, or cloud storage like Google Drive).
  4. The file is saved as passvault_backup_YYYYMMDD_HHMMSS.json.
Important: The backup file contains all your passwords in readable JSON format. Store it somewhere safe and do not share it. Anyone with the file can read your passwords.

Restoring from a backup

  1. Go to Settings → tap IMPORT BACKUP.
  2. Browse to your backup .json file and select it.
  3. Pass Vault reads the file and imports all accounts.
  4. A message tells you how many accounts were restored and how many were skipped.
Safe by design: Restore is additive — it never deletes existing data. If an account with the same site and username already exists, it is left untouched and counted as "skipped".
13

Security Features

Pass Vault is built with several layers of protection to keep your data private.

Encrypted Database

All data is stored using SQLCipher encryption. The key is generated randomly and kept in Android's secure key store — unreadable without your device credentials.

Biometric on Launch

The app cannot be opened without fingerprint, face, or PIN authentication every single time.

Biometric on Reveal

Even after you are inside the app, revealing a password requires a second biometric check.

Session Timeout

The app auto-locks after a configurable idle period, even when running in the background.

Screenshot Protection

The password detail screen is blocked from screenshots and hidden in the Recent Apps view.

Clipboard Auto-Clear

Copied passwords are automatically erased from the clipboard after 30 seconds.

No Internet Access

Pass Vault has no network permission at all. Your data cannot be sent anywhere — ever.

No Cloud Backup

Android's automatic backup is disabled. Your vault never leaves the device through the OS.

Tips for staying safe

Back up regularly
If you reset your device without a backup, your vault is gone — there is no recovery. Export a backup whenever you add important passwords.
Protect the backup file
The JSON file is not encrypted on its own. If you store it in cloud storage, make sure that storage account is well protected.
Use the password generator
The built-in generator creates strong 16-character passwords. Use it every time you create or update an account.
Set sensible reminders
Banking & email: 1–3 months. Social media: 3–6 months. Less critical accounts: 6–12 months.
